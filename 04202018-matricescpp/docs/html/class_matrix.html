Matrix is a class constructor.

It allocates dynamically a two-dimensional array that stores the matrix to be processed.

Parameters
_nlis the amount of lines of the array
_ncis the amount of columns of the array
  • If _nl or _nc is zero, the matrix will have all its dimensions defined to zero.
  • If the dimensions are negative, the program will be terminated.
